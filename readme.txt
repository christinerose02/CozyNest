== Description ==
Coazy-Nest is a charming e-commerce platform designed to make booking your perfect staycation simple, cozy, 
and convenient. Whether you're planning a quick weekend escape or a peaceful retreat, Coazy-Nest offers a 
curated selection of homestyle getaways with seamless browsing, real-time availability, and secure booking. 
With features like personalized recommendations, guest reviews, and exclusive local experiences, Coazy-Nest 
turns every stay into a memorable nest of comfort and relaxation.

== Installation ==
1. Download the Cozy Nest theme zip file.
2. Navigate to your website's admin panel.
3. Go to Themes > Upload Theme and select the downloaded file.
4. Click Install and then Activate the theme.
5. Now your ready to customize your website.
